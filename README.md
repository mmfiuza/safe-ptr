# SafePtr (work in progress)
A C++ container type.

## Compile and run examples
```
rm -rf build && cmake -S . -B build && cmake --build build && ./build/basic-usage
```

## Compile and run tests
```
rm -rf build && cmake -S . -B build && cmake --build build && ./build/test-all
```
